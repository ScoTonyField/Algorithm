## Leetcode 21 合并两个有序链表

将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

**示例：**

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

**方法：**

```python
# 有序链表合并-哨兵结点
   	def mergeTwoLists(l1,l2):
        soldier = ListNode(0)
        p = soldier
        while ( l1 != None and l2 != None ):
            if ( l1.val < l2.val ):
                p.next = l1
                l1 = l1.next
            else:
                p.next = l2
                l2 = l2.next
            p = p.next       
        if (l1 != None):
            p.next = l1
        if (l2 != None):
            p.next = l2
        return soldier.next
```
